Parágrafos e quebras de linha

Você pode escrever um parágrafo do jeito que voce quiser. Basta colocar tudo no meio das tags p e /p

Você viu que independente de enter, o texto não quebra linha. Lembrando que HTML é uma linguagem de marcação,
logo precisa de uma tag para isso. vou quebrar linhas desse parágrafo, usando a tag br
e ai, percebeu?

Mas se você quiser fazer parecer o sinal maior que ou menor quebra
você precisará colocar < e o ; dessa forma aparecerá os sinais da marcação
de quebra de linha no texto dentro do parágrafo.
Para criar uma linha horizontal usamos: <hr>
Para quebrar linha: <br>
E dessa forma aparecerá no texto os sinais de < e >

Para incluir comentários: <!-- 'texto aqui' -->
