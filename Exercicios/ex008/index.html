Texto marcado - Cor diferente

Obs.: HTML é semântica, cores são formas. Logo, esse procedimento deve ser realizado no CSS.

Como mudar a cor de um marcador:

Podemos mudar a cor de um texto marcado


Podemos criar um seletor para <mark> e padronizarmos a cor de todo texto que tiver mark, da seguinte forma:

<style>
mark {
background-color: lime;
}
</style>
